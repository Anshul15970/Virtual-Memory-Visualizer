# Virtual Memory Visualizer

An interactive web-based simulator to understand core **Operating System memory management concepts** including:

* Paging (FIFO, LRU, Optimal)
* Demand Paging
* Segmentation
* Page Fault Analysis
* Algorithm Comparison

 **Live Demo:**
https://anshul15970.github.io/Virtual-Memory-Visualizer/

---

# Features

## 🔹 Paging Simulation

* Supports **FIFO, LRU, and Optimal** algorithms
* Step-by-step execution
* Play / Pause animation
* Frame visualization
* Hit & Fault statistics
* Matrix view (textbook style)

---

## 🔹 Demand Paging

* Simulates **Disk → RAM page loading**
* Visual memory map
* Page fault tracking
* Step-by-step execution with animation

---

## 🔹 Segmentation

* Custom segment table (Base + Limit)
* Logical → Physical address translation
* Segmentation fault detection
* Memory layout visualization

---

## 🔹 Algorithm Comparison

* Compare FIFO, LRU, Optimal side-by-side
* Fault rate visualization
* Belady’s Anomaly detection
